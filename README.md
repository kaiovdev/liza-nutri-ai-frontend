# 🥗 Liza A.I (FRONTEND)

**Liza A.I** é uma aplicação inteligente construída com **Next.js**, **React**, **TypeScript** e **TailwindCSS**, capaz de gerar uma **dieta completa de 7 dias** totalmente personalizada com base nas suas informações, restrições e objetivo (perda de peso, ganho de massa ou manutenção).

Combinando **inteligência artificial**, **validações modernas**, **animações fluidas** e uma interface rápida e intuitiva, a Liza oferece uma experiência prática e eficiente para quem busca melhor alimentação.

---

## ✨ Tecnologias Principais

* ⚛️ **React 19**
* ▶️ **Next.js 16**
* 🟦 **TypeScript**
* 🎨 **Tailwind CSS**
* 🔁 **React Hook Form + Zod**
* 🎭 **GSAP e @gsap/react**
* 🧩 **Radix UI** (Label, Select, ScrollArea, Slot)
* 🔤 **Lucide React** (ícones)
* 📝 **React Markdown**
* 🎛️ **Tailwind Merge, CVA e clsx**

---

## 🧠 Sobre o Projeto

A **Liza A.I** coleta dados essenciais do usuário, como:

* Idade
* Peso
* Altura
* Nivel físico
* Objetivo

Com essas informações, o sistema gera automaticamente um **plano alimentar completo de 7 dias**, contendo:

* Café da manhã
* Almoço
* * Lanches
* Jantar

Tudo organizado e pensado para o seu objetivo.

## 📦 Instalação e Configuração

### **1. Clone o repositório**

```bash
git clone https://github.com/seuusuario/liza-ai.git
cd liza-ai
```

### **2. Instale as dependências**

```bash
npm install next react react-dom typescript tailwindcss postcss autoprefixer
npm install gsap @gsap/react
npm install react-hook-form zod @hookform/resolvers
npm install @radix-ui/react-label @radix-ui/react-select @radix-ui/react-scroll-area @radix-ui/react-slot
npm install lucide-react clsx class-variance-authority tailwind-merge
npm install react-markdown
```

Ou, se preferir:

```bash
yarn
```

```bash
pnpm install
```

### **3. Execute o ambiente de desenvolvimento**

```bash
npm run dev
```

### **4. Acesse no navegador**

```
http://localhost:3000
```
